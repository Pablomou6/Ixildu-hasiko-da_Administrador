# ProyectoBDII_Administrador

Para añadir ambos .jar (o de postgre mais o de cifrado de contraseñas), o que se debe facer é:  
	1-Visualizamos a carpeta "lib" dentro de netbeans co proyecto aberto  
	2.-Xa debería de haber algo estilo "JDK 17"  
	3.-Click dereito sobre a carpeta e damoslle en "añadir .jar"
	4.-Bsucamos o .jar que queremos e añadímolo
